# Trinity Consultant - Recruitment Website

A modern, accessible, and fully responsive recruitment firm website built with React and deployed on GitHub Pages.

## 🌟 Features

- ✅ **Fully Responsive Design** - Works on all devices
- ♿ **Accessibility First** - WCAG AA compliant
- 🎨 **Modern UI** - Beautiful gradient designs and smooth animations
- ⚡ **Fast Performance** - Optimized React build
- 📱 **Mobile Optimized** - Hamburger menu and touch-friendly interface
- 🔍 **SEO Ready** - Semantic HTML and meta tags
- 🚀 **GitHub Pages Ready** - One-command deployment

## 🎯 Sections

1. **Navigation Header** - Sticky nav with mobile menu
2. **Hero Section** - Compelling call-to-action
3. **Services** - 6 core recruitment services
4. **Testimonials** - Client success stories with ratings
5. **Contact Form** - Professional inquiry form
6. **Footer** - Company info and links

## 🚀 Quick Deployment

```bash
# 1. Install dependencies
npm install

# 2. Deploy to GitHub Pages
npm run deploy
```

**Live Site**: https://2025Subhashis.github.io/trinityconsultant/

## 📋 Tech Stack

- **React 18** - UI framework
- **CSS3** - Styling with flexbox and grid
- **gh-pages** - GitHub Pages deployment
- **Create React App** - Project scaffolding

## 🔧 GitHub Pages Setup

1. Go to **Settings → Pages**
2. Select **main** branch as source
3. Select **/ (root)** as folder
4. Click **Save**
5. Run `npm run deploy`
